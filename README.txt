PROBLEMA DE MINIMIZAR EL EXTREMISMO DE UNA POBLACIÓN

Se desarrolla un modelo de optimización que permite la reducción del extremismo en una población. Para ello, se han organizado los siguientes directorios:

1. Proyecto.mzn
   Contiene la implementación del modelo desarrollado.

2. ProyectoGUIFuentes
   - gui.py: archivo principal de la interfaz gráfica.
   - controlador.py: archivo que maneja la lógica de control.

3. DatosProyecto
   Contiene las baterías de pruebas utilizadas para el proyecto.

4. MisInstancias
   Contiene 9 archivos de pruebas propias diseñadas por el equipo.

INSTRUCCIONES DE USO

1. Clonar el repositorio y dirigirse al archivo gui.py.

2. Existen dos formas de ejecutar la interfaz:
   - Abrir una terminal y ejecutar el comando:
     python main.py
   - Usar un editor de código y correr el archivo main.py (usualmente marcado con un símbolo de triángulo ▶).

3. En la interfaz gráfica deberá:
   3.1 Presionar el botón "Cargar archivo", donde podrá cargar un archivo .txt desde el directorio DatosProyecto o MisInstancias.
   3.2 Presionar el botón "Obtener resultados", seleccionar el directorio donde desea guardar la entrada en formato .dzn. 
        Después de esto, el resultado aparecerá en la parte inferior de la interfaz.
